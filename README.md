# Daily Habit Tracker PWA

A progressive web app to track your daily schedule and build discipline.

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to vercel.com, click "Add New Project"
3. Import your GitHub repo
4. Framework: Vite (auto-detected)
5. Click Deploy

## Add to Home Screen

### iPhone
1. Open the deployed URL in Safari
2. Tap the Share button
3. Tap "Add to Home Screen"
4. Tap "Add"

### Android
1. Open the deployed URL in Chrome
2. Tap the three-dot menu
3. Tap "Add to Home Screen"

## Features
- 15 daily habits from your schedule
- Streak tracking
- 4-week calendar heatmap
- Per-habit completion rates
- Push notifications at key times
- Works offline
- System dark/light mode
